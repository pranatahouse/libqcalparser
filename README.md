libQCalParser
============
iCalendar parser written using Qt


===== Build Dependencies =====
Requires libqt4-dev on K/Ubuntu

===== Build Instructions =====

1) Make a seprate build dir :
    mkdir build

2) Change into the build dir :
    cd build

3) Run QMake :
    qmake ../

4) Run make :
    make

5) [Optional] Install using make install :
    sudo make install
